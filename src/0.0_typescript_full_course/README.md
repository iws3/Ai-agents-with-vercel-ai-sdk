## Ì≤ª Learning Environment

### Quick Start

```bash
# Navigate to Learners folder
cd Learners

# Install dependencies (Node.js must be installed)
npm install

# Compile TypeScript
npm run build

# Watch mode (auto-compile on changes)
npm run dev

# Run compiled code
npm start
```
