# Chapter 1: Foundations - Learner Exercises

This folder contains interactive exercises to practice Chapter 1 concepts.

## How to Use

1. **Open the exercise file** (e.g., `part-1-why-typescript/exercise-basic-types.ts`)
2. **Read the TODO comments** explaining what to do
3. **Write your code** replacing the ???
4. **Save the file**
5. **Compile** with: `npm run build` (from Learners/ directory)
6. **Fix TypeScript errors** shown in terminal
7. **Run** with: `npm start`
